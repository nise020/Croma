**이팩트**

1) 스킬 이팩트 추가(이명규 작업중)
* 에셋은 있으므로 테이블 수정이 필요

2) 사망 이팩트 추가(이명규 작업중)
* 에셋은 있으므로 테이블 수정이 필요

3) 몬스터가 스킬 사용시 스킬, 버프 이팩트 추가(이명규 작업중)
* 버프 시스템을 확인할 필요가 있음

**캐릭터 관련 작업**
0) 플레이어 사망 이벤트

1) 보스 몬스터 프리팹
* 기존에 캐릭터의 스케일을 키워서 프리팹을 한개씩 더 만들기
* 여유가 된다면 신규 에셋으로 추가(에셋은 찾음)

2) 보스 몬스터 Ai 수정(이명규 작업중)
* 전략패턴과 스킬 테이블의 추가로 구조가 바뀌어 그에 맞게 수정이 필요
* 사용하지 않는 기능 삭제(애니메이션 및 조건문 삭제)
* 스킬이 전부 쿨타임 중이면 가만히 있거나 추적 모드로 들어가서 테이블 값 또는 AI 조정이 필요

3) 보스 몬스터 출현 i(한태현 작업중)
* 타이머 or 레벨 달성 으로 계획중

4)보스,일반 애니메이터 작업(이명규 작업중)
* 이벤트 추가 등

5) 몬스터 아이템 드랍
* 퀘스트 아이템이 들어갈 경우
**ui 관련**

1) 스탯 분배Ui
* Ui 이미지 교체 및 Save 버튼 삭제 i(한태현,이명규 작업중)

2) 퀵슬롯 Ui(한태현 작업중)

3) 스테이지 클리어시 Ui(한태현 작업중)
* 페이드 효과로 대체중
4) 마지막 스테이지 클리어시 Ui

5) 플레이어 사망시 Ui 또는 연출
* 페이드로 대체 가능함

6) 플레이어 스킬 쿨타임 UI(이명규 작업중)
* 스킬은 작업이 되 있음
* 필살기 Ui 쪽은 아직 작업이 필요함


7) 마지막 스테이지 클리어시 Ui
* 점수 or 클리어 타임을 띄울수 있는 Ui 필요

8) 아이템 획득 UI 필요
* 기존 UI활용

9) 스킬 트리 UI
* 테이블 작업 필요(스킬의 설명이 필요 할거 같아서 Dec 포함)
